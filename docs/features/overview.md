# Features Overview

goDrive covers file management across web, Android, iOS, WebDAV, and CLI surfaces.

## Feature Support Matrix

See [Feature Support](../feature-support.md) for the full cross-platform matrix.

## Surface Summary

| Surface | Status |
|---|---|
| Web UI (Svelte) | Production-ready |
| Android app (Flutter) | Production-ready |
| iOS app (Flutter) | Production-ready |
| WebDAV | Supported |
| CLI (`godrive` binary) | Partial |

## Key capabilities

- [Web UI](web-ui.md) — browser-based file manager
- [Mobile Apps](mobile.md) — Android + iOS
- [Previews & Thumbnails](previews.md) — images, RAW, video, PDF, Office
- [Webhooks](webhooks.md) — event-driven integrations
